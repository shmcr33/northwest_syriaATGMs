# Northwest Syria ATGMs
This work-in-progress web map shows the use various anti-tank guided missiles (ATGMs) by a number of opposition factions in northwest Syria from 2014 to 2022.  The data used for this project is from geolocated videos of missile launches with relevant information stripped for viewing in the map when clicking on points.

As of 2/10/2026, this map contains geolocated point data from the following Syrian Opposition Groups:

Central Division
First Coastal Division
Jaish al-Izza
Tajumma Suqour al-Ghab
Liwa Suqour al-Jabal
Harakat Hazzm
13th Division
Liwa Fursan al-Haqq 

Addtional data will be uploaded as the geolocation and validation of videos from different groups is completed.
